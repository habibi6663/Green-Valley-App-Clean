# 🎓 Green Valley School Portal

A complete school management system built for seamless communication between Admin, Teachers, and Students.

---

## 🚀 Features

### 👨💼 Admin
- Manage users (teachers, students)
- Create notices and events
- Upload gallery images
- Full control over system

### 👨🏫 Teacher
- Upload notes (PDF, PPT, etc.)
- Mark attendance
- Assign homework
- Manage multiple classes

### 👨🎓 Student
- View notes and study materials
- Check homework
- View notices and events
- Access gallery
